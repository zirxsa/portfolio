# Muhammad Haziq — Portfolio

Personal portfolio site for Muhammad Haziq bin Mohd Azhar, a Bachelor of Information
Technology (Game Technology) undergraduate at Universiti Teknikal Malaysia Melaka (UTeM).

Built with plain HTML, CSS, and vanilla JavaScript — no build step, no framework.

## Structure

- `index.html` — page content
- `css/styles.css` — styling
- `js/main.js` — mobile nav toggle + footer year

## Running locally

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Deployment

Deployed via GitHub Pages from this repository.
